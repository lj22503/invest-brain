# 白酒行业研究框架

## 核心财务特征

| 指标 | 特征 |
|------|------|
| ROE | 高（15%-30%） |
| 毛利率 | 高（60%-90%） |
| 净利率 | 高（20%-40%） |
| 资产负债率 | 较低（20%-40%） |
| 总资产周转率 | 0.2-0.6次 |

高ROE主要来自**高净利率**（非高杠杆或高周转）。

## 牛股筛选标准

1. **年复合收益率** — 上市满三年，计算年化收益
2. **ROE稳定性** — 连续5年ROE>15%
3. **毛利率趋势** — 稳步提升的才是真龙头
4. **预收款/合同负债** — 蓄水池，越高越好

## 行业周期

| 阶段 | 时间 | 特征 |
|------|------|------|
| 第一阶段 | 05-07年 | 高端引领，茅五老窖表现优异 |
| 第二阶段 | 09-11年 | 中档地产酒崛起 |
| 第三阶段 | 11-12年 | 次高端发力 |
| 调整期 | 2013年 | 三公消费限制，行业低谷 |
| 复苏期 | 2015年后 | 商务和个人消费主导，弱复苏 |

## 竞争格局

### 高端白酒（千元以上）
- 贵州茅台（绝对龙头）
- 五粮液
- 泸州老窖

### 次高端（300-1000元）
- 洋河股份
- 山西汾酒
- 剑南春
- 郎酒

### 中高端（100-300元）
- 古井贡酒
-口子窖
- 今世缘

### 大众白酒（100元以下）
- 牛栏山
- 老白干

## 贵州茅台分析要点

1. **产品体系**：飞天茅台 + 系列酒（茅台酱香酒、赖茅等）
2. **量价策略**：茅台酒产量有限，批价坚挺
3. **金融属性**：收藏、礼赠、投资需求
4. **渠道改革**：i茅台直营平台，直销占比提升

## 关键指标监控

| 指标 | 观测频率 | 预警阈值 |
|------|----------|----------|
| 飞天茅台批价 | 周 | 跌破指导价 |
| 库存水平 | 月 | 渠道库存>3个月 |
| 动销增速 | 季 | 连续2季负增长 |
| 预收款变化 | 季 | 环比下降30%+ |

## 投资时钟

- **旺季**（中秋、春节前1-2月）：动销数据验证
- **淡季**：估值切换，布局次年
- **调整期**：三公消费政策、宏观经济下行

## 研究数据来源

- 茅台批价：酒业媒体（微酒、酒说等）
- 产量数据：中国酒业协会
- 财报：季报、年报
- 渠道调研：一线经销商访谈
