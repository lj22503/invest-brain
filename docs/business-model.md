## 一、策略总纲

**路线**: GitLab 开源商业化路径 — 开源核心（AGPLv3）+ 付费增值（云端服务）

**一句话**: InvestBrain 的本地 MCP Server + 基础 RAG 永远免费开源，付费点为「云端行为分析报告」——把用户本地做不到的事情做成 SaaS。

## 二、免费层设计

| 维度 | 范围 | 理由 |
|------|------|------|
| **本地 MCP Server** | 完全免费、无限制 | 开源核心，最大化社区传播 |
| **基础 RAG（16位大师）** | 完全免费、无限制 | 核心差异化壁垒，必须免费让人用 |

| **提醒推送** | 免费（飞书/钉钉/Bark） | 多一次推送 = 多一次产品曝光 |

## 四、开源协议

**选择: AGPLv3**

| 协议 | 防云厂商白嫖 | 社区友好度 | 商业化友好度 | 选择 |
|------|:---:|:---:|:---:|:---:|
| MIT | ❌ | ★★★★★ | ★★★★★ | |
| Apache 2.0 | ❌ | ★★★★ | ★★★★ | |
| **AGPLv3** | ✅ | ★★★ | ★★★ | ✅ |
| BUSL | ✅ | ★★ | ★★★ | |

**理由**: AGPLv3 是 GitLab/MongoDB 验证过的路径——云厂商不能直接拿开源代码做 SaaS 竞品，但普通用户和开发者不受影响。如果有人认为太严格，可以补充一个「AGPLv3 + 商业授权例外」条款。
